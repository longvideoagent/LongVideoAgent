# Training

`main` documents the current default training path for LongVideoAgent. An experimental training stack based on the updated `verl` is kept on the `newversion` branch for users who want to try it separately. Because configs, runtime behavior, and training outcomes may differ across branches, switch to the target branch first and then follow that branch's documentation.

Training documentation is organized into dedicated pages in the left navigation:

- `GRPO`
- `Offline Grounding Cache`
- `LoRA Adapter Merge`

Related dataset preparation steps are grouped under the top-level `Dataset Processing` section.
